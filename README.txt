===========================================
Author: Taihao Zhang

The Pap framework is a PHP application framework for building web applications
oriented towards


 System requirements
------------------------
The default framework requires the following components to function:

 - Apache 2
 - Apache mod_rewrite
 - PHP 5
 - PHP extensions
   - SPL (on by default, always on in PHP 5.3)
